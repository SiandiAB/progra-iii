## 🚀 Cómo agregar una nueva clase

1. Creá una carpeta nueva en `clases/`, por ejemplo `clases/clase-02/`.
2. Copiá el `index.html` de una clase anterior como plantilla.
3. Modificá el contenido de los slides (el CSS y JS ya están compartidos).
4. Agregá la tarjeta en el `index.html` raíz.
5. Hacé commit y push — GitHub Pages se actualiza solo.
